# Easy Dashboard

`Easy Dashboard` — это плагин для Obsidian, который генерирует красивую домашнюю страницу с быстрыми действиями, недавними заметками, папками и тегами.

Он упаковывает всю логику в готовый плагин, чтобы людям не приходилось вручную копировать длинные блоки `dataviewjs`.

## Возможности

- Генерирует домашнюю страницу-дэшборд
- Показывает приветствие в зависимости от времени
- Добавляет быстрый поиск
- Добавляет кнопки быстрых действий
- Показывает недавние заметки
- Показывает папки в виде кнопок
- Показывает теги в виде кнопок
- Генерирует отдельные страницы для просмотра папок и тегов
- Добавляет команды для открытия и обновления дэшборда
- Может подключаться к плагину `Homepage`

## Требования

- `Dataview` обязателен
- `Templater` опционален
- `Homepage` опционален

## Установка

### Установка вручную

1. Скачайте `manifest.json`, `main.js` и `versions.json`
2. Создайте папку `easy-dashboard` внутри `.obsidian/plugins/`
3. Поместите эти файлы в папку плагина
4. Включите плагин в разделе community plugins в Obsidian

### Установка через GitHub или BRAT

Можно опубликовать репозиторий и устанавливать плагин через BRAT или через zip-архив релиза.

## Настройка

1. Включите `Dataview`
2. Откройте `Settings -> Easy Dashboard`
3. Укажите пути к заметкам и папке inbox
4. Нажмите `Generate`
5. Если вы используете `Homepage`, нажмите `Connect`

## Команды

- `Generate or update dashboard files`
- `Open dashboard`
- `Connect dashboard to Homepage plugin`

## Примечание

Плагин генерирует обычные markdown-заметки с `dataviewjs`, поэтому пользователи всё ещё могут вручную менять внешний вид после установки.

## Чеклист релиза

1. Обновите версии в `manifest.json`, `package.json` и `versions.json`
2. Соберите zip-архив с файлами:
   - `manifest.json`
   - `main.js`
   - `versions.json`
3. Опубликуйте GitHub release
